# Changelog

All notable changes to Volt Client are documented in this file.

## 0.0.2 - 2026-07-10

### Added

- Local-first REST request editing and execution.
- Collections, folders, drag-and-drop organization, and native editor tabs.
- Query parameters, headers, multiple request body types, and common authentication methods.
- Global and environment-specific variables using `{{variable}}` syntax.
- Global State and workspace file storage modes.
- Formatted response bodies, response headers, status, duration, and responsive layouts.

### Security

- Documented that authentication values are stored with request data and must be reviewed before sharing file-based collections.
