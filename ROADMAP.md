# Roadmap

This roadmap communicates current product direction and does not guarantee delivery dates or final scope.

## Current priorities

- Improve stability and error reporting across common HTTP workflows.
- Expand documentation and examples for collections, environments, and file storage.
- Improve safeguards and guidance around credentials stored in request data.
- Refine accessibility, keyboard navigation, and theme compatibility.

## Under consideration

- Import and export workflows.
- Additional response viewers and request body formats.
- Request history and repeatable workflows.
- More granular secret-handling options.

Feature ideas and use cases are welcome through the repository's feature request form.
