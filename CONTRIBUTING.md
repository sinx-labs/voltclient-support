# Contributing to Volt Client Support

This repository accepts documentation feedback, bug reports, and feature requests. It does not accept source-code contributions for the proprietary Volt Client extension.

## Reporting bugs

Use the bug report form and provide the smallest reproducible example possible. Before submitting, remove credentials, cookies, private URLs, customer data, request and response bodies, and other confidential information.

## Requesting features

Describe the workflow or problem first, then explain the desired behavior. This helps evaluate alternatives without committing to a specific implementation prematurely.

## Documentation

Corrections to public documentation may be proposed through an issue. Do not submit internal documents, source code, credentials, or unpublished product information.
